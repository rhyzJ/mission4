## 👩🏼‍💻 Installation 👩🏼‍💻

1. **Clone the repository:**

    ```bash
    git clone https://github.com/rhyzJ/mission4.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd mission4
    ```

3. **Start the application using Docker Compose:**

    ```bash
    docker-compose up --build
    ```
